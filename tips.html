#OurSafetyTips


#1/ Use Strong & Unique Passwords

‣ Never use common patterns (abc123 etc), words (password etc) or names of pets or loved ones as passwords

‣ Use a combination of capital and lowercase letter along with number and symbols if possible (ChrisPlap2002$)

‣ Use strong password generators to generate a unique password for each site

‣ Use password managers to keep track of all your unique passwords (only use reputable services)

‣ Check the strength of your password everytime you create a new password

#2/ Use a Firewall

‣ A Firewall is a network security system which monitor in and outgoing traffic. It establishes a barrier between trusted and untrusted sources using predefined rules.

‣ Turn on Firewall in MacOS

‣ Turn on Firewall in Windows

‣ Turn on Firewall in Ubuntu

Note: Firewall settings may vary in each Linux distro, refer to documentation for your distro before tweaking the settings

#3/ Use an Antivirus

‣ An Antivirus software detects and removes viruses from your computer

‣ There are numerous antivirus programs. Some may require a subscription but most have a free version.

‣ Even if you use a paid antivirus program, it is crucial to keep it updated to ensure it can defend against new viruses

#/ Miscellaneous Tips

‣ Never keep personal devices unattended

‣ Use secure (HTTPS) sites

‣ Keep all software updated

‣ Do not click on suspicious links

‣ Be cautious when using public WiFi

   *WiFi Honeypot acts as a legitimate WiFi access point and can introduce malware onto your system
